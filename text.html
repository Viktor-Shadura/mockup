CSS властивість column-count розбиває вміст елемента на задану кількість стовпчиків.

Властивість CSS column-width встановлює ширину стовпчика у багатостовпчиковому макеті. У контейнері буде стільки стовпчиків, скільки може поміститися. Якщо ширина контейнера вже вказаного значення, ширина окремого стовпчика буде меншою за заявлену ширину стовпчика.

Властивість CSS column-gap встановлює розмір проміжку між стовпчиками елемента.
